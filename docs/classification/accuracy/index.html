    Confusion Matrix

    Many metrics for measuring performance of a classification model depends on the values in the confusion matrix.

    The basic confusion matrix is used to describe a binary classifier's performance on a test data set for which the true labels (classes) are known. It shows the number of correct and incorrect predictions made by the model, by considering the two classes separately. Therefore, the matrix has four cells containing the following numbers:

    • True Positives (TP): the number of Class 1 data points that are correctly predicted as Class 1
    • False Positives (FP): the number of Class 0 data points that are incorrectly predicted as Class 1
    • True Negatives (TN):
    • False Negatives (FN):

    Accuracy

    $$ \mathrm{Accuracy} = \frac{TP + TN}{TP + FN + FP + TN} $$

    The problem of accuracy

    Precision & Recall

    $$ \mathrm{Precision} = \frac{TP}{TP + FP} $$

    $$ \mathrm{Recall} = \frac{TP}{TP + FN} $$

    F1 Score

    $$ \mathrm{F1} = 2 \times \frac{\mathrm{Precision} \times \mathrm{Recall}}{\mathrm{Precision} + \mathrm{Recall}} $$
